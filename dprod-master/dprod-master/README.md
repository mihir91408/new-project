### Run Locally

To run **Dprod** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/dprod.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/dprod.git
```
